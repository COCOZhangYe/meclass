# 觅课堂接口文档

## 用户登录

http请求方式: POST

Content-Type：json

```http
/user/login 
```

| 字段  | 说明 | 类型    | 是否必填 |
| ----- | ---- | ------- | -------- |
| uname | 名字 | String  | 是       |
| pwd   | 密码 | String  | 是       |
| flag  | 标志 | Integer | 是       |

 

登录成功返回JSON数据包:

```json
 {
    "code": 0,
    "msg": "登陆成功 ",
    "data": {
        "iduser": 5,
        "uname": "hello",
        "pwd": null,
        "flag": 1
    }
}
```

用户或者密码为空返回JSON数据包:

```json
 {
    "code": 1,
    "msg": "用户或密码为空 ",
    "data": null
}  
```

用户或者密码错误返回JSON数据包: 

```json
{
    "code": 1,
    "msg": "用户或密码错误",
    "data": null
}    
```

| 参数 | 说明                              |
| ---- | --------------------------------- |
| msg  | 提示消息                          |
| Data | 返回登录的用户名和密码            |
| flag | 标志0:学生 标志1:教师             |
| code | 状态 1：登录失败 状态 0：登录成功 |

## 用户注册

http请求方式: POST

Content-Type：json  

```http
/user/register
```

| 字段  | 说明 | 类型    | 是否必填 |
| ----- | ---- | ------- | -------- |
| uname | 名字 | String  | 是       |
| pwd   | 密码 | String  | 是       |
| flag  | 标志 | Integer | 是       |

注册返回JSON数据包： 

```
{
    "code": 0,
    "msg": "注册成功",
    "data": null
}
```

用户注册失败返回JSON数据包：

```
{
    "code": 1,
    "msg": "注册失败",
    "data": null
}
```

| 参数 | 说明                         |
| ---- | ---------------------------- |
| msg  | 提示消息                     |
| code | 状态码 1：失败，状态 0：成功 |

## 评论 - `/comment`

### 查询评论 - `/comment/get`

请求参数：

| 字段    | 必须 | 类型   | 说明   |
| ------- | ---- | ------ | ------ |
| videoId | 是   | Number | 视频ID |

响应：

*如果查询成功响应的是一个数组，每个元素的结构如下*

| 字段             | 必须 | 类型    | 说明                                 |
| ---------------- | ---- | ------- | ------------------------------------ |
| commentId        | 是   | Number  | 评论ID，唯一                         |
| username         | 是   | String  | 用户名                               |
| commentMsg       | 是   | String  | 评论内容                             |
| replyId          | 否   | Number  | 回复的评论ID，如果有                 |
| createTime       | 是   | Number  | 发布的UNIX时间戳                     |
| edited（待实现） | 是   | Boolean | 是否被编辑过                         |
| like（待实现）   | 是   | Array   | 点赞这条评论的用户，每个元素是用户名 |
| pId              | 否   | Number  | 父id(如果有回复)                     |

响应示例：

```
{
  "code": 0,
  "msg": "ok",
  "data": [
    {
      "commentId": 1267,
      "username": "example",
      "comment": "Good video.",
      "timestamp": 1593865719,
      "edited": false,
      "like": ["example2"]
    },
    {
      "commentId": 1269,
      "username": "example2",
      "comment": "This is a reply comment. edited.",
      "timestamp": 1593865937,
      "edited": true,
      "reply": 1267,
      "like": []
    }
  ]
}
```

目前响应成功

```json
{
    "code": 0,
    "msg": "查询成功",
    "data": [
        {
            "commentId": 1,
            "username": "coco",

            "replyId": null,
            "commentMsg": "第一条评论",
            "createTime": "2020-07-05 22:41:21",
            "pid": null
        },
        {
            "commentId": 2,
            "username": "coco1",
            "videoId": 1,
            "replyId": null,
            "commentMsg": "第二条评论",
            "createTime": "2020-07-05 23:01:25 ",
            "pid": null
        }
    ]
}
```

查询失败

```json
{
    "code": 1,
    "msg": "查询失败",
    "data": null
}
```
